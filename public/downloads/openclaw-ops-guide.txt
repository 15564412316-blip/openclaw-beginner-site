OpenClaw 后续使用指南（必备）

1. 首次安装完成后做什么
- 打开 API Key 配置页面，准备并填写你的模型密钥。
- 推荐先做一次最小测试，确认模型调用成功。

2. API Key 获取与配置
- 你可自行选择任意合规模型提供商。
- 获取 Key 后，在 OpenClaw 的配置文件中填写。
- 请勿把 API Key 发给他人或上传到公开仓库。

3. 如何验证运行是否成功
- 执行启动后，观察是否有报错。
- 若报错，请复制错误信息并提交工单。

4. 后续如何再次运行
- 进入安装目录（默认：~/openclaw 或 %USERPROFILE%\\openclaw）。
- 按项目文档中的启动命令运行服务。

5. 常见问题
- 网络失败：检查网络或镜像源设置。
- 依赖安装失败：先确认 Node/npm 版本，再重试。
- 配置失败：确认 Key 是否有效，是否有多余空格。

6. 支持与服务边界
- 自动安装订单仅支持一次脚本下载。
- 若安装失败且无法修复，可提交工单。
- 如需进一步代办服务，请按站点说明购买对应服务。
